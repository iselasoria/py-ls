#fundamentals 
Precedence in Python is the order in which expressions get evaluated.
Python follows this order:

| Operators                                                     | Meaning                          |
| ------------------------------------------------------------- |:--------------------------------:|
| `()`                                                          | parentheses                      |
| `**`                                                          | exponent                         |
| `*`                                                           | multiplication                   |
| `/`                                                           | division/modulo                  |
| `+`                                                           | addition                         |
| `-`                                                           | subtraction                      |
| `==`, `>`, `>=`, `<`, `<=` , `is`, `is not`, `in`, `not in`,  | comparison, identity, membership |
| `not`, `and`,`or`                                             | logical                          |
